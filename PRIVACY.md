**Picklist Dependency Manager**
Effective Date: 13 February 2026

Picklist Dependency Manager is a Chrome extension designed to help Salesforce administrators and developers manage dependent picklist configurations using CSV.

**Data Collection**

This extension does not collect, store, sell, or transmit personal data.
It does not use analytics, tracking tools, advertising services, or third-party data processors.

**Salesforce Data Access**

The extension accesses Salesforce configuration data only when the user actively uses it. All operations occur locally within the userâ€™s authenticated Salesforce browser session.
No Salesforce data is transmitted to external servers.

**Data Storage**

The extension does not store credentials, access tokens, or metadata outside the active browser session. No persistent external storage is used.

**Permissions**

Permissions are requested only to access Salesforce pages necessary to export and update dependent picklist configurations. The extension does not access unrelated websites.
