# Picklist Dependency Manager

Picklist Dependency Manager is a Chrome extension that allows Salesforce administrators and developers to export and import picklist dependency mappings using CSV.

## Support

If you encounter any issues or would like to request a feature:

- Open an Issue in this repository
- Or contact: shroffaayushi25@gmail.com

When reporting issues, please include:
- Salesforce edition
- Browser version
- Steps to reproduce
- Screenshot (if applicable)

## Privacy

This extension does not collect, store, or transmit data to external servers.  
All processing occurs locally within your authenticated Salesforce session.
